== Description ==

Plugin gratuito para integração com o Banco Inter para pagamentos via Boleto Bancário e Pix com retorno automático.

== Third-Party Services ==

Este plugin depende dos seguintes serviços de terceiros:

1. **Banco Inter OAuth Service**
   - **Service URL:** [Banco Inter OAuth](https://cdpj.partners.bancointer.com.br/oauth/v2/token)
   - **Terms of Use:** [Banco Inter Terms of Use](https://cdpj.partners.bancointer.com.br/terms)
   - **Privacy Policy:** [Banco Inter Privacy Policy](https://cdpj.partners.bancointer.com.br/privacy)

O plugin comunica-se com estes serviços nas seguintes circunstâncias:
- Ao obter um token OAuth.
- Ao acessar recursos protegidos usando o token OAuth.

Ao utilizar este plugin, você concorda com os termos de uso e políticas de privacidade dos serviços mencionados.

== Installation ==

= Automatic installation through WordPress =

1. Select **Add New** in your WordPress on the **Plugins tab**.
2. Look for **Banco Inter MWP for WooCommerce**.
3. Click on **Install** Now and **Activate Plugin**.

= For manual installation via FTP: =

1. Upload the **mwp-gateway-banco-inter** to the **/wp-content/plugins/** folder.
2. Activate it in the **Plugins** section of your WordPress.
